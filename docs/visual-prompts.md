# Multi-cloud Backup FinOps Dashboard Prompts

"Generate a high-fidelity dashboard visualization for enterprise backup spend. The UI should feature a master dark-mode theme with glassmorphism cards. Key elements: A massive 'Current Month Spend' count of $145,820, a 'Waste Detected' alert showing $32,450 in red. Include a multi-colored line chart showing spend trends across Azure (blue), AWS (orange), and GCP (green). Add a section for 'Top 5 Savings Opportunities' with progress bars."

## 2. Global Resilience Map
"Visualize a 3D global isometric map representing multi-region backup vaults. Pulse animations connecting East US to West Europe representing data replication. Overlay cost bubbles over each continent showing per-region spend. Stylistic choice: Cyber-noir aesthetic with neon blue data lines and clean typography."

## 3. Storage Tier Heatmap
"A 2D tree-map visualization of data tiers. Large blocks representing 'Hot' tier (High cost, high access), medium blocks for 'Cool', and small dots representing 'Archive'. Use a gradient from hot-orange to cold-blue. The visual should clearly represent how moving data from Hot to Archive reduces visual volume and cost."
